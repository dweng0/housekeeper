# Zigbee2MQTT gateway adapter

## What to build

Implement `Zigbee2MqttGateway`, a concrete adapter for the `DeviceGateway` port that speaks the zigbee2mqtt MQTT convention. Wired into the composition root as the default gateway.

The `DeviceGateway` port is unchanged — this is purely an adapter concern.

## Acceptance criteria

- [ ] `Zigbee2MqttGateway` implements `DeviceGateway`
- [ ] `publish(topic, payload)` sends to `zigbee2mqtt/<topic>/set` with JSON payload `{"state": <payload>}` (or raw if payload is already valid JSON)
- [ ] `subscribe(topic, handler)` listens on `zigbee2mqtt/<topic>` for state updates
- [ ] Adapter wired into composition root as the `DeviceGateway` implementation
- [ ] MQTT broker host/port configurable via `AppConfig`

## Implementation notes

Zigbee2MQTT topic convention:
- Command: `zigbee2mqtt/<friendly_name>/set` → `{"state": "ON"}`
- State read: `zigbee2mqtt/<friendly_name>` → `{"state": "ON", "linkquality": 45, ...}`

`Device.topic` stores the zigbee2mqtt friendly name — adapter constructs full topic path internally. Nothing above the adapter needs to know about zigbee2mqtt.

Future non-zigbee devices (Z-Wave, Matter) get their own adapter implementing the same `DeviceGateway` port.

## Out of scope

- Auto-discovery via `zigbee2mqtt/bridge/devices`
- Device pairing / management
- Handling zigbee2mqtt-specific metadata (linkquality, battery etc)

### Steps

1. **Create `src/mqtt/zigbee2mqtt-gateway.ts`**:
   ```ts
   import type { DeviceGateway } from "../ports.js";
   import type { SharedMqttClient } from "./mqtt-client.js";

   export function createZigbee2MqttGateway(client: SharedMqttClient): DeviceGateway {
     return {
       async publish(topic, payload) {
         const fullTopic = `zigbee2mqtt/${topic}/set`;
         let mqttPayload: string;
         try {
           JSON.parse(payload); // already valid JSON
           mqttPayload = payload;
         } catch {
           mqttPayload = JSON.stringify({ state: payload });
         }
         await new Promise<void>((resolve, reject) =>
           client.publish(fullTopic, mqttPayload, (err) => (err ? reject(err) : resolve()))
         );
       },
       subscribe(topic, handler) {
         const fullTopic = `zigbee2mqtt/${topic}`;
         client.subscribe(fullTopic);
         client.on("message", (t, msg) => {
           if (t === fullTopic) handler(msg.toString());
         });
       },
     };
   }
   ```
